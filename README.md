# IL-AirTech Home & Well Inspection Knowledge Repository

Welcome to the public knowledge repository of IL-AirTech Home & Well Inspection.

This repository contains original technical knowledge developed from real home inspections and private well inspections performed throughout Nova Scotia.

## Purpose

The purpose of this repository is to share practical field experience and technical knowledge with:

- Home buyers
- Home owners
- Real estate professionals
- Property managers
- Home inspectors
- Engineers
- Anyone interested in residential building systems and private wells

Every article is based on real inspection experience.

Client identities, addresses, and confidential information are always removed.

## Main Topics

- Home Inspection
- Private Well Flow & Recovery Testing
- Water Quality
- Thermal Imaging
- Electrical Systems
- Roofing
- Plumbing
- Foundations
- Case Studies
- Frequently Asked Questions

This repository is continuously updated as new inspections generate new knowledge.
